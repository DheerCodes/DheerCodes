## 🚀 Projects

### 🧊 [dryice](https://github.com/DheerCodes/dryice) — Home Automation Hub
A phone-first home automation hub deployed on Vercel. No dedicated server, no always-on laptop.

- 📺 Samsung TV control via SmartThings API (OAuth + rotating token via GitHub Gist)
- ❄️ LG AC control for multiple rooms via LG ThinQ API
- 🌤️ Live weather dashboard
- 🔑 Fully automatic OAuth token reset from phone — no laptop needed
- Built with Node.js · Python · Vercel Serverless

---

### 📇 Contact Sync — Google Apps Script
A multi-account Google Contacts sync system across 6 Gmail accounts using a shared Google Sheet as the sync database.

- Syncs contacts bidirectionally across family accounts
- Runs on a schedule via Google Apps Script triggers
- Zero infrastructure — lives entirely in Google's ecosystem
- Built with Google Apps Script · Google Sheets API
