## 🚀 Projects

### 🧊 [dryice](https://github.com/DheerCodes/dryice) — Home Automation Hub
A phone-first home automation hub deployed on Vercel. No dedicated server, no always-on laptop.

- 📺 Samsung TV control via SmartThings API (OAuth + rotating token via GitHub Gist)
- ❄️ LG AC control for multiple rooms via LG ThinQ API
- 🌤️ Live weather dashboard
- 🔑 Fully automatic OAuth token reset from phone — no laptop needed
- Built with Node.js · Python · Vercel Serverless

---

### 🔗 [go-url](https://github.com/DheerCodes/go-url) — Personal URL Shortener
A fast, password-protected URL shortener running entirely on Cloudflare's free tier. No server, no database bill.

- 🔀 Custom slugs including full path support — e.g. `go.yourdomain.com/tv/ch1.m3u8`
- 📊 Click tracking on every redirect
- ✏️ Edit slug, destination URL, and title without losing click history
- 🔒 Password-protected dashboard — API auth enforced server-side
- 🔄 Auto-deploys from GitHub via Cloudflare Pages git integration
- Built with Vanilla JS · Cloudflare Pages · Workers KV

---

### 📡 [zunox](https://github.com/DheerCodes/zunox) — Personal Indian IPTV EPG
A self-maintaining XMLTV EPG feed for Indian IPTV channels, built to fill gaps in standard EPG sources.

- 📋 Merges JioTV + TataPlay AIO EPG sources, deduplicates and filters to personal channel list
- 🔄 GitHub Actions cron rebuilds `epg.xml` every 6 hours automatically
- 🗑️ Stale programme data filtered out on every build — only today onwards
- 📺 Live status dashboard at [zunox.vercel.app](https://zunox.vercel.app) — shows channel coverage, current programmes, last Action run
- Built with Node.js · GitHub Actions · Vercel Static Hosting

---

### 📇 Contact Sync — Google Apps Script
A multi-account Google Contacts sync system across 6 Gmail accounts using a shared Google Sheet as the sync database.

- Syncs contacts bidirectionally across family accounts
- Runs on a schedule via Google Apps Script triggers
- Zero infrastructure — lives entirely in Google's ecosystem
- Built with Google Apps Script · Google Sheets API
